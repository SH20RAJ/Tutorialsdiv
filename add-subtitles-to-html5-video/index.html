Add Subtitles to HTML5 Video






Sample Code - 
  
<video controls>
<source src="https://www.iandevlin.com/html5test/webvtt/v/upc-tobymanley.theora.ogg" type="video/ogg">
   <track label="English" kind="subtitles" srclang="en" src="sample.vtt" default>
   <track label="Deutsch" kind="subtitles" srclang="de" src="sample.vtt">
   <track label="Español" kind="subtitles" srclang="es" src="https://www.iandevlin.com/html5test/webvtt/upc-video-subtitles-en.vtt">
</video>
